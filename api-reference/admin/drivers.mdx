---
title: "List Drivers"
description: "List all drivers with filtering by status and performance metrics"
openapi: "GET /api/admin/drivers"
---

Get a list of all drivers with their statistics.

## Authorization

<Info>Requires ADMIN role. Include JWT token in the Authorization header.</Info>

```
Authorization: Bearer <token>
```

## Query Parameters

| Parameter   | Type    | Default | Description             |
| ----------- | ------- | ------- | ----------------------- |
| `page`      | number  | 1       | Page number             |
| `limit`     | number  | 10      | Items per page          |
| `search`    | string  | -       | Search by name or email |
| `available` | boolean | -       | Filter by availability  |

## Example

```bash
# Get all drivers
curl "https://api.trash2treasure.lk/api/admin/drivers" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."

# Filter available drivers
curl "https://api.trash2treasure.lk/api/admin/drivers?available=true" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## Notes

- Only users with DRIVER role are returned
- `available` filters drivers with less than 5 active bookings
- `stats` include lifetime performance metrics
- `currentBookings` shows active (non-completed) bookings
