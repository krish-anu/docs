---
title: "Admin Dashboard"
description: "Get admin dashboard overview with user, driver, and booking statistics"
openapi: "GET /api/admin/dashboard"
---

Get dashboard statistics for administrators.

## Authorization

<Info>Requires ADMIN role. Include JWT token in the Authorization header.</Info>

```
Authorization: Bearer <token>
```

## Example

```bash
curl "https://api.trash2treasure.lk/api/admin/dashboard" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## Notes

- Only accessible by users with ADMIN role
- Statistics are calculated in real-time
- `monthlyStats` shows current year data
- `topDrivers` shows top 5 drivers by completed pickups
