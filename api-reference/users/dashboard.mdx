---
title: "Dashboard"
openapi: "GET /api/users/dashboard"
---

Get dashboard statistics for the current user.

## Authorization

<Info>
  Requires authentication. Include JWT token in the Authorization header.
</Info>

```
Authorization: Bearer <token>
```

## Example

```bash
curl https://api.trash2cash.lk/api/users/dashboard \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## Notes

- Statistics are calculated from all-time user data
- `recentBookings` returns the last 5 bookings
- `monthlyStats` shows data for the current year
- Amounts are in LKR (Sri Lankan Rupees)
