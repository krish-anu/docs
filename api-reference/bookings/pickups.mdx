---
title: "List Pickups"
description: "List available pickup schedules and time slots for waste collection"
openapi: "GET /api/bookings/pickups"
---

Get all pickup locations or pending pickups. This endpoint is used primarily by drivers to view awaiting pickups.

## Authorization

<Info>
  Requires authentication. Include JWT token in the Authorization header.
</Info>

```
Authorization: Bearer <token>
```

## Query Parameters

| Parameter | Type   | Description                 |
| --------- | ------ | --------------------------- |
| `status`  | string | Filter by booking status    |
| `limit`   | number | Number of results (max 100) |
| `offset`  | number | Pagination offset           |
