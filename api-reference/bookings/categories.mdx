---
title: "Waste Categories"
openapi: "GET /api/bookings/categories"
---

Get all available waste categories with current pricing based on market rates.

## Authorization

<Info>This endpoint is public. No authentication required.</Info>

## Example

```bash
curl "https://api.trash2treasure.lk/api/bookings/categories"
```

## Notes

- Only active categories are returned
- Prices are in LKR (Sri Lankan Rupees) per kilogram
- `minPriceLkrPerKg` and `maxPriceLkrPerKg` represent the price range
- Use the `id` when creating a booking
- Admins can update pricing via `/api/admin/pricing`
