---
title: "Get Booking By ID"
description: "Get detailed information about a specific waste pickup booking in Trash2Treasure"
openapi: "GET /api/bookings/{id}"
---

Get detailed information about a specific booking.

## Authorization

<Info>
  Requires authentication. Include JWT token in the Authorization header.
</Info>

```
Authorization: Bearer <token>
```

## Path Parameters

| Parameter | Type   | Description  |
| --------- | ------ | ------------ |
| `id`      | string | Booking UUID |

## Example

```bash
curl https://api.trash2treasure.lk/api/bookings/booking-uuid \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## Notes

- Users can only view their own bookings
- Drivers can view bookings assigned to them
- Admins can view all bookings
