---
title: "Create Booking"
description: "Create a new waste pickup booking for the Trash2Cash marketplace"
openapi: "POST /api/bookings"
---

Create a new waste pickup booking.

## Authorization

<Info>
  Requires authentication. Include JWT token in the Authorization header.
</Info>

```
Authorization: Bearer <token>
```

## Request Body

```json
{
  "wasteCategoryId": "category-uuid",
  "estimatedWeightRange": "5-15kg",
  "addressLine1": "123 Main Street",
  "city": "Colombo",
  "postalCode": "05",
  "scheduledDate": "2024-02-01T10:00:00Z",
  "scheduledTimeSlot": "10:00-12:00",
  "specialInstructions": "Please call before arriving"
}
```

## Parameters

| Field                  | Type   | Required | Description                           |
| ---------------------- | ------ | -------- | ------------------------------------- |
| `wasteCategoryId`      | string | Yes      | UUID of waste category                |
| `estimatedWeightRange` | string | Yes      | Weight range (e.g., "5-15kg")         |
| `addressLine1`         | string | Yes      | Pickup street address                 |
| `city`                 | string | Yes      | City name                             |
| `postalCode`           | string | Yes      | Postal/ZIP code                       |
| `scheduledDate`        | string | Yes      | ISO 8601 date                         |
| `scheduledTimeSlot`    | string | Yes      | Time slot range (e.g., "10:00-12:00") |
| `specialInstructions`  | string | No       | Additional instructions               |
