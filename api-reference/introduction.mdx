## Overview

The Trash2Treasure API provides programmatic access to the waste marketplace platform. Use the API to integrate booking management, user authentication, and administrative functions into your applications.

## Base URL

```
https://api.trash2treasure.lk
```

## Authentication

All API requests require authentication using JWT tokens. Include the token in the Authorization header:

```
Authorization: Bearer <your-jwt-token>
```

## Response Format

All responses are in JSON format:

```json
{
  "success": true,
  "data": { ... },
  "message": "Optional message"
}
```

## Error Handling

Error responses include an error code and message:

```json
{
  "success": false,
  "error": "UNAUTHORIZED",
  "message": "Invalid credentials"
}
```

## Rate Limiting

API requests are limited to 1000 requests per hour per IP address.

## SDKs and Libraries

- **JavaScript/TypeScript**: Official SDK available on npm
- **Python**: Community-maintained library
- **Postman Collection**: Available for testing

## Getting Started

1. Sign up for an API key at [app.trash2treasure.lk](https://app.trash2treasure.lk)
2. Authenticate using your credentials
3. Start making requests to the API
