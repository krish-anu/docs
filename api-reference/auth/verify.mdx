---
title: "Verify Email"
description: "Verify user email address with OTP for account completion"
openapi: "POST /api/auth/verify"
---

Verify user email with the OTP sent during signup.

## Request Body

```json
{
  "email": "user@example.com",
  "otp": "123456"
}
```

## Parameters

| Field   | Type   | Required | Description              |
| ------- | ------ | -------- | ------------------------ |
| `email` | string | Yes      | Email used during signup |
| `otp`   | string | Yes      | 6-digit OTP from email   |

## Response

<Tabs>
  <Tab title="200 OK">
    ```json
    {
      "message": "Email verified successfully",
      "isVerified": true
    }
    ```
  </Tab>
  <Tab title="400 Bad Request">
    ```json
    {
      "statusCode": 400,
      "message": "Invalid OTP"
    }
    ```
  </Tab>
  <Tab title="404 Not Found">
    ```json
    {
      "statusCode": 404,
      "message": "User not found"
    }
    ```
  </Tab>
</Tabs>

## Example

```bash
curl -X POST https://api.trash2treasure.lk/api/auth/verify \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "otp": "123456"
  }'
```

## Notes

- OTP expires after 10 minutes
- Upon successful verification, email is marked as verified
- User can now login normally with their credentials
- Use `/api/auth/resend-otp` if OTP expires
