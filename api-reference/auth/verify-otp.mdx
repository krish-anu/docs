---
title: "Verify OTP"
description: "Verify user email address using the OTP sent during signup"
openapi: "POST /api/auth/verify-otp"
---

Verify a user's email and account using the OTP sent during signup or password reset.

## Request Body

```json
{
  "email": "user@example.com",
  "otp": "123456"
}
```

## Parameters

| Field   | Type   | Required | Description              |
| ------- | ------ | -------- | ------------------------ |
| `email` | string | Yes      | Email used during signup |
| `otp`   | string | Yes      | 6-digit OTP from email   |

## Response

<Tabs>
  <Tab title="200 OK">
    ```json
    {
      "message": "Email verified successfully",
      "user": {
        "id": "user-uuid",
        "email": "user@example.com",
        "role": "CUSTOMER"
      }
    }
    ```
  </Tab>
  <Tab title="400 Bad Request">
    ```json
    {
      "statusCode": 400,
      "message": "Invalid or expired OTP"
    }
    ```
  </Tab>
  <Tab title="404 Not Found">
    ```json
    {
      "statusCode": 404,
      "message": "User not found"
    }
    ```
  </Tab>
</Tabs>

## Example

```bash
curl -X POST https://api.trash2treasure.lk/api/auth/verify-otp \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "otp": "123456"
  }'
```

## Notes

- OTP is valid for 15 minutes
- OTP can only be used once
- Check your email if you don't receive the OTP
