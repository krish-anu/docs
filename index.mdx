## What is Trash2Cash?

Trash2Cash is a comprehensive waste management platform that connects users with drivers for recyclable waste pickup. Users can schedule pickups, track their earnings, and contribute to a cleaner environment.

## Key Features

- **User Authentication** - Secure login with JWT and OTP verification
- **Booking Management** - Create, track, and manage waste pickup bookings
- **Real-time Notifications** - Push notifications for booking updates
- **Admin Dashboard** - Comprehensive admin panel for platform management
- **Driver App** - Dedicated interface for drivers to manage pickups
- **Pricing Management** - Dynamic pricing based on waste categories
- **AI Assistant** - Built-in chat assistant for user support
